Ohjelmarunko

Jokainen ohjelma tarvitsee ohjelmarungon, jonka avulla kääntäjä osaa kääntää ohjelman tietokoneen suoritettavaksi.

Tämän materiaalin jokaisen ohjelman runkona voit käyttää alla olevaa main.cpp tiedostoon sijoitettavaa runkoa:

#include "suomi.hpp"

kokonaisluku paaohjelma()
{
  palauta 0;
}

Yllä oleva ohjelmarungon ensimmäinen rivi #include "suomi.hpp" lataa Suomi-CPP määritykset, jotta ohjelmassa voidaan käyttää suomenkielisiä avainsanoja.
