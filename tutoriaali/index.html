Suomi-CPP tutoriaali

Mitä ja miksi?

Suomi-CPP on suomeksi käännetty version C++ ohjelmointikielestä. Sen tarkoituksena on helpottaa tekstipohjaisen ohjelmoinnin aloittamista varsinkin lasten keskuudessa. Kääntämällä ohjelmointikielen avainsanat suomeksi englantia osaamattomien pienempien lasten on toivon mukaan helpompi lähestyä ohjelmointia.

Kielen avainsanat on käännetty (toistaiseksi osittain) ja lisäksi syötteen lukemista ja konsoliin tulostamista on helpotettu apuluokkien avulla.

Miten?

Suomi-CPP:n lähdekoodi sekä ohjeet sen käyttöönottoa varten löytyvät Githubista.

Ohjelmakoodia kirjoittaaksesi tarvitset tekstieditorin tai ohjelmointiympäristön. Esimerkiksi Visual Studio Code on toimiva vaihtoehto.

Kirjoitettu lähdekoodi tulee vielä kääntää konekieliseksi, jotta se on tietokoneen suoritettavissa. Tätä varten tarvitset kääntäjän, kuten g++.
